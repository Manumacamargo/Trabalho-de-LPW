# Music Tracker

O Music Tracker é uma aplicação web desenvolvida para gerenciar e organizar músicas pessoais. Ele permite criar categorias, adicionar músicas, buscar, filtrar e excluir músicas de forma simples e intuitiva. A aplicação foi construída usando HTML, CSS, Bootstrap 5 e JavaScript, com armazenamento de dados local usando localStorage.

## Funcionalidades

* Gerenciamento de Categorias

* Adicionar novas categorias com nome e cor personalizada.

* Visualizar categorias em dropdowns nos formulários de música.

# Gerenciamento de Músicas

* Adicionar músicas com título, artista e categoria.

* Visualizar lista de músicas existentes.

* Filtrar músicas por categoria.

* Buscar músicas por título ou artista.

* Excluir músicas com confirmação em modal (botões "Cancelar" e "Deletar").

# Persistência de Dados

* Todas as categorias e músicas são salvas no localStorage do navegador, garantindo que os dados permaneçam entre sessões.

* Interface Responsiva

* Layout otimizado para diferentes tamanhos de tela usando Bootstrap 5.

* Cards interativos com efeitos de hover e cores de categoria destacadas.
